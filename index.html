SignSpeech: A Communication Device for Sign Language Users

Collaborators: me, Oliver Fang

We built a wearable glove that translates American Sign Language letters and displays them through an iOS application. This was on of my favorite projects because it gave me hands on experience on establishing reliable wireless communication between devices. The basic operation of the system is as follows: the gloves sends sensor data to an iOS device via Bluetooth, the iOS device echoes the data to a server via TCP connection, the server translate the data via a machine learning algorithm and sends the translation to iOS app.
